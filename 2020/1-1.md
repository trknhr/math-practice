# 【1級】　1次：計算技能検定

## 問題1．4次方程式

次の方程式を考える。

$$
x^4 + x^3 + kx^2 + 8x + 6720 = 0 \quad (k は定数)
$$

4個の解のうち、2個の解について積を求めたところ **80** となりました。  
このとき、$k$ の値を求めなさい。

## 解答


解を以下のように定義
$$\alpha, \beta, \gamma, \delta$$

$$\alpha * \beta = 80$$

4次元方程式

$$
x^4 + a * x^3 + b * x^2 + c * x + d = (x - \alpha)(x - \beta)(x - \gamma)(x - \delta)
(x - \alpha)(x - \beta) = x^2 - (\alpha + \beta)x + \alpha * \beta
(x - \gamma)(x - \delta) = x^2 - (\gamma + \delta)x + \gamma * \delta

(x^2 - (\alpha + \beta)x + \alpha * \beta)(x^2 - (\gamma + \delta)x + \gamma * \delta)
= x^4
    - (\alpha + \beta + \gamma + \delta) * x^3 
    + (\alpha * \beta + \alpha * \gamma + \alpha * \delta + \beta * \gamma + \beta * \delta + \gamma * \delta) * x^2
    - (\alpha * \gamma * \delta + \beta * \gamma * \delta + \gamma *  \alpha * \beta +  \delta * \alpha * \beta)
    + \alpha * \beta * \gamma * \delta
$$

所与の数値を入れると

$$
\alpha + \beta + \gamma + \delta = -1
$$ 

$$
\alpha * \beta + \alpha * \gamma + \alpha * \delta + \beta * \gamma + \beta * \delta + \gamma * \delta = k
$$

$$
\alpha * \gamma * \delta + \beta * \gamma * \delta + \gamma * 80 + \delta * 80 = - 8
$$

$$
80 * \gamma * \delta = 6720
$$

$$
\alpha * \gamma * \delta + \beta * \gamma * \delta + \gamma * 80 +  \delta * 80 = - 8
$$

$$
80 * \gamma * \delta = 6720
$$
$$
80 * \gamma * \delta = 6720
$$
$$
=> \gamma * \delta  = 84
$$

$$
\alpha * \gamma * \delta + \beta * \gamma * \delta + \gamma * 80 +  \delta * 80 = - 8
$$
$$
=> 84 * \alpha + 84 * \beta + \gamma * 80 +  \delta * 80 = -8
$$
$$
=> 84 * \alpha + 84 * \beta + 80 * (-1 - \alpha - \beta) = -8
$$
$$
=> -80 + 4 * \alpha + 4 * \beta = -8
$$
$$
=> \alpha + \beta = 18
$$

$$
=> \alpha + \frac{80}{\alpha} = 18
$$

$$
=> \alpha^2- 18\alpha + 80 = 0
=> (\alpha - 8)(\alpha - 10) = 0
$$


$$
\gamma + \delta = -1 - (\alpha + \beta)
$$

$$
\gamma + \delta = -19
$$

$$
\gamma + \frac{84}{\gamma} = -19
=> \gamma^2 + 19\gamma + 84 = 0
=> (\gamma + 12)(\gamma + 7) = 0
$$

よってそれぞれの答えは

$$
8, 10, -12, -7
$$

解がわかったので

$$
\alpha * \beta + \alpha * \gamma + \alpha * \delta + \beta * \gamma + \beta * \delta + \gamma * \delta = k
$$

$$
=> 80 + 8*-12 + 8 * -7 + 10 * -12 + 10 * -7 + -12 * -7 = k
$$

$$
k = -178
$$
